# 🔍 Network Port Scanner GUI

A Python-based **Network Port Scanner with GUI** that allows users to scan open ports on a target system and identify running services. This project demonstrates practical concepts of **networking, cybersecurity, and socket programming**.

---

## 🚀 Features

* Scan open ports on a target IP or domain
* User-friendly GUI built with Tkinter
* Supports custom port range scanning
* Multithreading for faster performance
* Basic service detection (HTTP, FTP, SSH, etc.)
* Displays real-time scan results

---

## 🛠️ Technologies Used

* **Python** – Core programming language
* **Socket Programming** – For port communication
* **Tkinter** – GUI development
* **Threading** – Faster scanning
* **TCP/IP Networking** – Core networking concepts

## ▶️ How to Use

1. Enter the **Target IP Address / Domain**
2. Specify the **Port Range** (e.g., 20–100)
3. Click on **Scan**
4. View the list of **Open Ports and Services**
